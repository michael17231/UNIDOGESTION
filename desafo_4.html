🧩¡SUERTE! Tú puedes

Reconocimiento de Emociones Faciales en Tiempo Real con la Máquina Educable en mBlock.

Diseñar un programa en mBlock que permita reconocer las emociones de un usuario a través de gestos faciales utilizando los bloques de la Máquina Educable para clasificar las expresiones faciales en categorías, como felicidad, tristeza, sorpresa y enojo. Utilizar la cámara del dispositivo para captar las expresiones faciales del usuario y usar los bloques de la Máquina Educable para entrenar un modelo con ejemplos de gestos faciales para cada emoción, solicitando al usuario que realice expresiones que correspondan a cada una de las categorías, luego utilizar los bloques de predicción para clasificar las emociones que detecte en tiempo real mostrando en pantalla la emoción identificada y proporciona una respuesta  auditiva según la emoción detectada, por ejemplo, un sonido suave si se detecta sorpresa, permitir que el usuario repita el ejercicio realizando diferentes gestos faciales para probar la precisión del sistema y ajustar el modelo en caso necesario.
